# IGMP Snooping

## Use case
- This is a requirement for PXE boot.
- This configlet enables IGMP snooping for any virtual network with `mcast` in the VN name.

## Scope of application

| Role              | Yes/No |
|-------------------|--------|
| Spine/SuperSpine  | No     |
| Leaf              | Yes    |
| Access-Switch     | No     |
